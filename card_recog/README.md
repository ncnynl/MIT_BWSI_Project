# Card Recognition
This Alexa skill can identify a standard playing card

#### Installation

As of now, this skill has no intents
###### Intent Schema (with placeholder intent) 
```
{
  "intents": [
    {
      "intent": "pokerIntent"
    }
  ]
}

```
###### Sample Utterances (with placeholder intent)
```
pokerIntent poker
```
